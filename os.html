Windows 1.0

Windows 1.0 is a graphical personal computer operating system developed by Microsoft. It provides an environment which can run graphical programs designed for Windows, as well as existing MS-DOS software.

Windows 95

Windows 95 is a consumer-oriented operating system developed by Microsoft. It merged Microsoft's formerly separate MS-DOS and Windows products. It featured significant improvements over its predecessor, Windows 3.1, most notably in the GUI and in its simplified "plug-and-play" features.

MS-DOS

MS-DOS is an operating system for x86-based PCs mostly developed by Microsoft. Collectively, MS-DOS, its rebranding as IBM PC DOS, and some operating systems attempting to be compatible with MS-DOS, are sometimes referred to as "DOS".

Play

Windows 98

Windows 98 is a graphical operating system. It is the second major release in the Windows 9x line of operating systems and the successor to Windows 95. Like its predecessor, Windows 98 is a hybrid 16-bit and 32-bit monolithic product with the boot stage based on MS-DOS.
